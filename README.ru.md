### 🔭 Текущий фокус: Экосистема Codex

В данный момент я полностью сосредоточен на развитии **[Codex Hub](https://codexdlc.github.io/codex-docs/)** — модульного open-source Python стека, созданного для ускорения разработки надежных production-ready приложений. Он состоит из множества независимых, но отлично работающих вместе библиотек:

* **`codex-core` & `codex-platform`**: Фундамент — DTO, настройки, фоновые воркеры, потоки событий и работа с Redis.
* **`codex-django` & `codex-services`**: Интеграция среды выполнения и разделение высокоуровневой бизнес-логики для масштабируемых Django-бэкендов.
* **`codex-ai`**: Независимые от провайдеров абстракции LLM и чистые интерфейсы интеграции ИИ.
* **`codex-bot`**: Продвинутый фреймворк на базе Aiogram для эффективной автоматизации Telegram-продуктов.

Я верю в Clean Architecture, Dependency Injection и создание инструментов, которые делают работу разработчиков удобной (как, например, **`codex-django-cli`**).
